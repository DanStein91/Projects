Project: Data Visualization
Dataset description:

Ireland Primary Schools.

This project is entirel done with json/VegaLite. The visualization bellow shows the primary schools in Ireland. Each dot represents a school, the colour represents whethere it is a boys, girls, or mixed school, and the size shows the number of students. This graph is interactive, one can choose the Sex, Province, and select a group of number of students.
